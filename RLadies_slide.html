Polish Reports: What to Present?

  • An example:

We here define a function

\[ f(x) = x + 1, \]

and the code to implement this function is

my_function = function(x) {
  return (x + 1)
}
  • Another example:

We here define a function \(f(x) = x + 1\), and the code to implement this function is my_function = function(x) {return (x + 1)}.
